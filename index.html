1. You have to guess a number between 1 to 10 (mutually inclusive)
  2. You will find three chances to guess the correct number
  3. If your given number is greater then correct number you will find a hint that your given number is greater
  4. If your given number is smaller then correct number you will find a hint that your given number is smaller
  5. If your given number is exactly equal to the correct number you will find a hint that your given number is correct
  6. After availing three chances if you can not make the correct guess you will be looser
  7. By clicking the bellow button the game will be started
